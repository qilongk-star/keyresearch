🧪 About RefCheck

RefCheck is developed by Dr.Key using AI Agent technology. The entire application — from UI design to backend logic — is built and iterated by an AI coding assistant.

Currently open-source on GitHub for transparency and collaboration. Primarily intended for XJTLU internal use. If you find it useful, feel free to share with colleagues.

⚠️ Disclaimer: This tool assists with reference checking but does not guarantee 100% accuracy. Always verify critical references manually. Results may vary depending on document formatting and reference style.
